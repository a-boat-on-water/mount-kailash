# 11. 对宗教、文化的尊重，负责任的旅行

## 对宗教、文化的尊重

冈仁波齐是宗教圣地，不建议在转山途中喝酒。现在转山沿途各补给点的茶馆、宾馆均不售卖酒类或含酒精的饮品。

路上若遇到佛塔、转经轮、玛尼石墙、明显的玛尼石堆这些，除非是本教徒，请尽可能遵循藏传佛教的传统，从左侧通过或顺时针绕行。

如果想拍摄别人，请先征得对方的允许。转山途中遇到的藏族人，特别是年纪大的，可能汉语能力有限甚至听不懂。用你的身体语言和微笑去表达你的善意请求，拍完之后回放给对方看或者把照片发给对方，以表示你的谢意。如果对方不愿意，则不要强行拍照。

## 负责任的旅行

我连续十多年在冈仁波齐周边行走，转山路沿途的垃圾问题见过不少。一些缺乏环保意识的转山者将各种难以降解的垃圾随手扔到路边。由于海拔高、气候恶劣、维护困难，转山路上产生的垃圾不能像在城市里那样被快速清理掉，尤其在转山旺季，容易出现路边垃圾堆积过多未能及时处理的情况。

特别是不要将塑料袋、糖果零食包装这类重量轻、难以降解的垃圾留在卓玛拉前后这一段，这段路海拔最高，车辆开不上去，垃圾运送基本靠人背马驮。由于冈仁波齐周边经常刮大风，轻垃圾容易被大风吹走，飘到远处后就无法清理。我曾在一些远离转山路、人迹罕至的地方都发现塑料袋，基本可以肯定是被风吹过去的。建议转山者随身带备垃圾袋，将重量轻难以降解的垃圾收集带回塔钦再抛弃到垃圾桶。如果转山途中看到严重的垃圾问题，可打景区综合服务热线400-6666-712告知情况，以便他们尽快处理。

转山中尽量少消费一次性包装的产品，例如塑料瓶的饮用水/饮料等，自带水杯/水瓶装水。

如果需要挂经幡，有可能请选择采用传统工艺、使用易降解材料制作的环保型经幡。

不要采摘沿途各种花草，高寒地区植物发育缓慢，你见到的美丽瞬间，它们可能需要数年来生长。

不惊吓不干扰野生动物，它们也是世世代代在冈仁波齐周边生活的神山子民。

更多的负责任户外旅行注意事项，请参照"无痕山野"（Leave No Trace）七项原则。
