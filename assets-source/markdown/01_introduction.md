# 1. 为文化旅行者、朝圣者而写的冈仁波齐转山指南

## 1.1 缘起

2008年9月，我开始了前后共40天的第一次西藏旅行。当来到冈仁波齐旁，却因大雪封山，转山计划无奈放弃。但幸运的是，回程路上终于见到了水晶塔般直耸云霄的冈仁波齐神山。在2012年秋天我的第二次西藏旅行中，完成了第一次冈仁波齐转山。

2014年6月，我启动了《朝圣冈仁波齐》人文专题长期摄影项目，从那时起每年都会在冈仁波齐神山旁待上一段长时间，以身体力行的方式一边转山一边拍摄，至2025年已持续了十二年。希望通照片表达对神明的敬畏、对信仰的钦佩、对生命的谦卑、对善良的赞美……

从2014年6月到2025年11月，这十二年间我在神山旁累计待了736天，已完成64次冈仁波齐转山。

> ▲ 野狗"阿黄"与本文作者Jansen Wong在冈仁波齐神山旁（2015年）

由于要为之后的摄影项目整理结集做准备，这些年也一直在广泛收集关于冈仁波齐神山的相关材料，积累了大量英、藏、汉文书籍、文章、地图、图片、视频档案，包括不少权威资料。

在收集中发现，冈仁波齐的中文（汉语）高质量信息非常少，自己手头的核心参考资料，超过80%都是非中文的。之前那些中文旅行指南书、旅游杂志文章里关于冈仁波齐转山的内容，或网上各种的冈仁波齐介绍文章、攻略，大部分都有很多严重错误或过时信息，并且原创内容比例极少，多是东抄西凑，经常以讹传讹。

而在近二十年来的中文信息世界里，"冈仁波齐转山"多被误解或刻意包装成户外徒步线路，由于普遍欠缺对相关历史文化、宗教信仰的认知了解，那些"冈仁波齐徒步攻略"基本只能将原本属于圣地朝拜的"转山"当成了一项户外运动来介绍。

因此在2019年初，我就开始准备资料，着手写一本与众不同、准确实用、图文并茂的《冈仁波齐转山朝圣指南》，将我多年多次的转山经验、以及对相关文化的学习所得融入其中，希望做出一本回归"转山"原来的朝圣、文化本质的实用指南。

## 1.2 为文化旅行者、朝圣者而写的冈仁波齐转山指南

那时流传于网上，或者是将网上资料整理印刷出版的中文冈仁波齐"转山攻略"，基本都以徒步爱好者的角度出发。他们更多着重于一个"挑战性"——全程50公里的转山路，有三分之一路段海拔都在5000米以上，而最高点——5650米的卓玛拉垭口，估计是不少人一辈子完全靠自己双腿能走上的最高海拔，所以在户外运动圈子就有些人渲染"征服的荣耀感"作为冈仁波齐转山这路线的卖点。

其实冈仁波齐的转山道，以户外徒步的角度看并不怎么"有趣"，被炒作成"N大徒步线路之一"纯属"美丽"的误会——除了冈仁波齐峰，沿途没有太多"美景"，全年大多时间，周边都可以用"荒凉"来形容。全程地貌变化不大，基本都是石头——看的是石头山，走的是乱石路。完全没有乔木只有低矮的灌木与草本，最高的植物还不到一米。难得草绿花开的7、8月又是雨季，冈仁波齐神山不易完整见到。而下完卓玛拉垭口，剩下的20公里转山路更是"枯燥无味"，没什么可再"挑战"，也无"好风景"可"提神"，感觉漫漫无期永远走不到尽头，所以部分徒步者就在这段路放弃，坐"救援车"回去了。

在这类徒步攻略里，往往将"距离、时间、海拔"作为内容的唯一重心，而忽略淡化甚至是完全不了解"转山"原本的圣地朝拜文化意义。这些年我在转山途中遇到很多来"徒步冈仁波齐"的人，基本都是默默走路，他们一路上询问、讨论最多的，不外乎"还有多远"、"还要多久"、"明天几点出发冲顶垭口"之类。

与那些偏重自然风光的徒步线路不同，**冈仁波齐转山路是一条已被世世代代藏地朝圣者走了数百年的转经道**，蕴含深厚的宗教意义与文化内涵。

**冈仁波齐转山与转佛塔、转寺庙、转圣城、转圣湖一样，藏语中称为"廓拉"སྐོར་ར或"聂廓"མཆོད་སྐོར，即"转经"的意思——以绕行圣物、圣地的仪式，来消除恶业、积累福德，是一种朝圣活动。**

冈仁波齐被认为是藏传佛教密宗五大本尊之一的"胜乐金刚"的居所，而冈仁波齐神山旁的每座山峰几乎都有名字，多数称之为某某神明的宫殿或雕像，转山路沿途还有大量的圣迹与典故，所以冈仁波齐周边是不折不扣的"众神之地"。有说法称，以冈仁波齐为中心，与四周的群山组成了一个天然的坛城，转一次神山也等同转了一圈"胜乐金刚"的坛城。

笃信业报轮回的朝圣者，视转一圈冈仁波齐是体验一次从今生到来世的过程，期望洗净今世罪业，积累福报，获得今生的安乐和更好的来世。

传统上，信众都会以步行的方式走完朝圣之路，认为这样获得的功德比骑马更大。而虔诚者甚至以15天一圈的磕长头方式转山，以求更能利益众生。

> ▲ 磕长头转冈仁波齐的朝圣者

我在2019年动笔写《冈仁波齐转山朝圣指南》时，转山者能参考的中文资料基本有两类：一类是各种旅游书籍杂志或各户外旅行网站、社交平台上流传的经过整理的"转山徒步攻略"，另一类则是各种网络平台上发表的个人转山经历游记。

那些"冈仁波齐转山徒步攻略"，虽然有一定条理化，但错误、过时信息非常厉害，严重误导后来者。

部分个人转山"游记"能提供一些当时的最新信息，但普遍是流水帐形式，不具系统性。而且多数游记作者都是第一次转山，本来就对当地情况不熟悉，相关的文化、宗教知识也了解有限，再加上现场的语言沟通障碍，导致游记中各种错误理解、以讹传讹的情况很常见，甚至有好事者自行加戏、肆意发挥……

无论是那些转山徒步攻略还是个人转山游记，都普遍缺乏对冈仁波齐转山途中各种宗教圣迹、相关文化的介绍。即便有也基本是摘抄几篇老文章上的文字，原文本来就写得模模糊糊，抄的人也只能复制粘贴一大堆名称，多数自己都不知道是什么意思、具体在什么地方。

而一些藏族学者用汉文所写的冈仁波齐文章，虽然综合翻译了藏文典籍的记述，但要么就属于学术论文形式，要么就写成散文美文，对各种圣迹并无准确清晰的位置描述，大多也缺少照片、地图的辅助，对于转山朝圣者的实用指导很有限。

在持续多年的转山期间，我为了摄影项目的拍摄到过冈仁波齐旁很多地点，既对周边环境相当熟悉，又对转山者的各种需求和关心事项非常了解。

**所以我要写的是一本针对文化旅行者、朝圣者，内容全面详细、准确实用、图文并茂的《冈仁波齐转山朝圣指南》（简称《转山指南》）。除了提供比之前各种攻略、游记详尽准确N倍的行程计划建议、转山路线详解外，更会着重对冈仁波齐转山途中各种宗教圣迹及相关文化的介绍。并全部配以自己的实拍照片，标示出各圣迹的确切位置，有图有真相，让大家再不需要瞎猜。**

**我希望这本《转山指南》能为那些兴趣重点在文化体验、圣地朝拜的转山者提供另一种视角，而不只是或只能是将冈仁波齐转山当成一项户外徒步活动。**

2019年8~10月，我在神山旁住了66天，除了转山、摄影项目拍摄，还专门花了大量时间做转山沿途圣迹的实地调查、采访，为写《冈仁波齐转山朝圣指南》收集第一手材料。2020年7~8月的一个来月中，我对转山路沿途及塔钦（转山者的"大本营"）进行了第二次专门的实地调查核实，同时采访请教多位熟悉冈仁波齐的学者老师、僧人，解决了不少之前留下的疑问，随后在2020年9月发布了第一版的《冈仁波齐转山朝圣指南》。

之后我每次再回到冈仁波齐神山旁，都会一边转山一边进行新的调查采访，为《转山指南》后续版本的内容补充与更新去采集一手素材。除了实地调研，我也持续收集整理更多的高质量相关资料进行学习，以丰富知识、拓宽视野。

我的目的是做一本内容严谨但通俗易懂的旅行指南书，而不是学术论文或散文小说，因此会以中性平实的语言去讲述，但也尽量写得不那么闷，让大家看得没那么累。

《冈仁波齐转山朝圣指南》是针对转山者（特别是初次转山者）的各种需求而系统化编排撰写，内容涵盖全面，8.0版有约8.4万字，140多幅实拍照片及原创地图。大家可根据自己的不同情况，选择阅读需要的章节。

8.0版《冈仁波齐转山朝圣指南》的各种信息内容，均来自截至2026年4月下旬，我对转山路全程及塔钦的多次详细实地调查所得，后续版本会持续更新。

## 1.3 冈仁波齐转山的权威指南

《冈仁波齐转山朝圣指南》中所有转山沿途圣迹的位置，都经过我多次的现场核实，同时参考多部专门讲述冈仁波齐的高质量、权威的藏英汉文书籍资料，并不断请教熟悉冈仁波齐文化的当地人、学者、僧人，与我实地调查收集的信息进行相互验证。也请多位受过良好教育、精通藏语的藏族朋友，为我解释、翻译较为复杂的藏文，尽最大可能提高内容的准确度。

十多年来，我凭着一颗赤子之心，完全融入当地社群并被充分认可，与当地民众及冈仁波齐周边寺庙建立了深厚的个人友谊，从而更容易获得各种关键信息。

《冈仁波齐转山朝圣指南》中关于各种圣迹解说的内容，除有特别备注外，来源均需不少于两个高可信度且独立的信息源才会采纳。一些暂时只有孤证，或者不同信息来源说法差异很大的都暂不收录，待我继续考证有了定论后再于新版本中补充。另外转山沿途一些圣迹故事在流传中衍生了多种诠释，基于篇幅有限，通常我都只选其中一种说法。还有部分圣迹则担心在详细公开、被广泛传播后对其资源造成过度索取，这类圣迹会故意不写。

我每次转山，除拍摄了海量照片作为影像资料外，同时都携带专门的GPS记录仪记录全程轨迹数据，拥有大量第一手的详细线路、距离、海拔的技术数据。本《转山指南》标出的转山途中走过的各处海拔，都是以我多次"肉身亲测"的GPS记录数据为基础，结合Google Earth卫星地图、SRTM高程数据进行综合微调，相信误差能在±10米内。

《冈仁波齐转山朝圣指南》自2020年的第1版起，对个人阅读就一直免费（具体条款请看第14章"版权声明与使用限制"），并保持每年至少发布一个新版本，更新信息、增添内容。5年多来，《冈仁波齐转山朝圣指南》已帮助大量读者圆满完成冈仁波齐转山，并以完全不同于"户外徒步运动"的角度去了解感受冈仁波齐周边的深厚文化。

《冈仁波齐转山朝圣指南》结构系统化、内容涵盖全面、图文并茂、准确实用、严谨可靠，是冈仁波齐转山解说的权威之作，获得读者、当地人、学者、文旅领导的高度评价，也成为众多导游、领队的讲解范本。2026年1月，我应西藏旅游股份有限公司阿里分公司邀请，以《冈仁波齐转山朝圣指南》为蓝本，为冈仁波齐转山路新设立的圣迹解说牌撰写了全部内容。
