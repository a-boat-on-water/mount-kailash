# Glossary of Key Religious & Cultural Terms

| 中文 | English | 释义 |
|------|---------|------|
| 转山 / 转经 | Kora / circumambulation | 围绕神山、寺庙、佛塔等圣地顺时针绕行一圈，是藏传佛教重要的修行与朝圣方式。 |
| 外转 / 外圈 | Outer kora | 围绕冈仁波齐峰一整圈的标准转山路，约 52 公里，本指南所述路线。 |
| 内转 / 内圈 | Inner kora / Nangkor | 特殊的内圈转经道，传统上需先完成 13 次外转才有资格走。极凶险，本指南不介绍。 |
| 萨嘎达瓦节 | Saga Dawa Festival | 藏历四月十五日，藏传佛教最重要的节日，纪念释迦牟尼诞生、成道、涅槃。通常在公历 5 月底~6 月中，期间转山朝圣者剧增。 |
| 马年 | Year of the Horse | 藏地传说佛祖诞生、成道于马年。马年转山功德相当于平常年份的 13 倍，朝圣活动特别隆重。下一个马年为 2026 年。 |
| 功德 | Merit | 通过善行、修行、朝圣等积累的精神福报，可净化业障、利益后世。 |
| 业障 / 罪业 | Karmic obscurations | 由过去恶行所造、阻碍解脱的业力。转山被认为可以净除业障。 |
| 磕长头 | Full-body prostration | 藏传佛教最虔诚的朝拜方式：每走一步即五体投地一次，以身体丈量大地。一些朝圣者以磕长头方式完成转山。 |
| 胜乐金刚 | Chakrasamvara | 藏传佛教密宗五大本尊之一。冈仁波齐被视为其「身坛城」。 |
| 经幡 | Prayer flags (lungta) | 印有经文图像的彩色布幡，风吹动一次即被视为念诵一遍。转山路沿途处处可见。 |
| 玛尼石 / 玛尼堆 | Mani stones | 刻有六字真言「嗡嘛呢叭咪吽」等经文的石头，朝圣者会沿路堆放或顺时针绕行。 |
| 塔钦 | Tarchen / Darchen | 藏语「大经幡杆」之意。位于冈仁波齐神山以南 10 公里的小镇，是绝大多数转山者的「大本营」、起点与终点。海拔约 4600 米。 |
| 卓玛拉 / 卓玛拉垭口 | Dolma La / Tara's Pass | 转山路全程最高点，海拔 5650 米，以度母命名。藏地朝圣者视通过此垭口为「重生」——前世/旧业死于山的一面，新生于另一面。 |
| 芝热寺 / 直热普寺 | Dirapuk Monastery | 冈仁波齐北壁正对的寺院，得名于「母牦牛角洞」（米拉日巴修行洞）。是转山第一天的住宿地。 |
| 尊珠寺 | Zutulpuk Monastery | 得名于「米拉日巴神变洞」（米拉日巴以神通将岩石压低修出的洞）。是转山第二天的住宿地。 |
| 曲古寺 | Chuku Monastery | 冈仁波齐西壁对面的寺院，供奉三圣物：曲古塑像、神奇白海螺、神奇石锅。 |
| 恰采岗 / 朝拜处 | Chaktsalgang / Prostration Point | 转山路上四个面对冈仁波齐的朝拜处，朝圣者会在此跪拜神山。分别位于冈仁波齐的西南、西、东、东南方位。 |
| 天葬台 | Sky burial ground | 天葬是藏地传统的葬礼方式。转山路上有「色雄天葬台（大天葬台）」和「斯巴恰天葬台（小天葬台）」等。朝圣者可在此象征性「献出」头发、衣物碎片或经历象征性死亡仪式。 |
| 业之镜 | Mirror of Karma | 卓玛拉垭口前的圣迹，相传可映照出转山者的善恶业。 |
| 善恶检验石缝 | Sin-testing rock crevice | 三块巨石之间窄窄的缝隙，相传罪孽深的人钻不过去。 |
| 甘露 / 圣水 | Amrita / sacred water | 被神山加持的水，沿途多处泉水、瀑布被视为甘露圣水，可净化身心、治病。 |
| 加持 | Blessing | 佛、菩萨、上师或圣地的力量注入，使物品或人获得神圣品质。 |
| 拉章 | Labrang | 寺庙大喇嘛的府邸。塔钦拉章（竹巴拉章）见证了冈仁波齐周边两百多年的特殊历史。 |
| 颇章 | Phodrang / palace | 宫殿。许多转山路沿途的山峰被命名为各种本尊或菩萨的「颇章」（宫殿），如「黑财神宫殿」、「黄财神宫殿」等。 |
